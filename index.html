Tags aprendidas

Tag h1 á h6

São usadas para definir títulos dentro do arquivo html, sendo á H1 está tag geralmente define o título principal da pagina , as tags como H2 são para definir os subtítulos até o h6 defino o título menos importante , é importante não pular os níveis de títulos comece pelo H1 e segue a sequência até h6.

Tag p

A tag P para definir um paragrafo no seu texto.

Tag mark

A tag mark define o texto que deve ser marcado ou destacado.

Tag small

A tag small define texto menores, como direitos autorais e outros comentários secundários.

tag i

A tag Idefine texto que deve ser mostrado em itálico.

Tag u

A tag U define texto que deve ser sublinhado.

Tag strong

A tag strongmarca o texto como negrito.

Tag ol

A tag OL é uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética. como por exemplo:

  1. Item 1
  2. Item 2
    1. item 1.3
    2. litem2.3

    Tag ul

    A tag UL uma lista não ordenada (com marcadores). como por exemplo:

    • item 1
    • item 2

      Tag li

      A tag LI define um item da lista.A tag é usada dentro de listas ordenadas ( ol ), listas não ordenadas ( ul ).Em ul , os itens da lista geralmente serão exibidos com marcadores.Em ol, os itens da ista geralmente serão exibidos com números ou letras.

      Tag a

      A tag A define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do(a) elemento é o href atributo, que indica o destino do link. Outro atributo importante é o target que especifica a janela do navegador de destino como por exemplo : Target _blanc abre uma nova aba no navegador. Target _self abre o link na mesma aba.

      Tag hr

      A tag HR traça uma linha indicando quebra temática dentro do html.

      Tag sub

      A tag sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O.

      Tag sup

      A tag supdefine texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como N º.

      Tag blockquote>

      A tag blockquote uma seção citada de outra fonte.

      Tag font

      A tag font usada no HTML 4 para especificar o tipo da fonte, o tamanho da fonte e a cor do texto.

      Tag del

      A tag delé usada para identificar o texto que foi excluído de um documento, mas retido para mostrar o histórico de modificações feitas no documento.

      Tag abbr

      A tag abbr define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
